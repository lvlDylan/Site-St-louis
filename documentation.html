Nom de domaine

Enregistrement, choix du nom et configuration DNS

Qu'est-ce qu'un nom de domaine ?

Un nom de domaine est l'adresse tapée dans le navigateur pour accéder au site. Exemple : bts-sio-chateaulin.fr. Il est loué auprès d'un bureau d'enregistrement (registrar).

Pour un lycée public, il est recommandé d'utiliser l'extension .fr ou .edu.fr. Le domaine ac-paris.fr est géré par le rectorat.

Cookies & RGPD

Le RGPD impose d'obtenir un consentement clair avant tout dépôt de cookie.

Logique JavaScript (scripts.js)

if (localStorage.getItem("cookieChoice")) {
  banner.style.display = "none";
  return;
}

accept.addEventListener("click", () => {
  localStorage.setItem("cookieChoice", "accepted");
  banner.style.display = "none";
});
